First: what is computational neuroscience?

It uses mathematics, coding, statistics, and models to understand how neurons, circuits, brains, and behaviour work. It can be close to biology, psychology, AI/ML, physics, engineering, or neuroimaging depending on the question.

If you like biology

Model neurons and circuits

Start with spikes, synapses, Hodgkin–Huxley, integrate-and-fire models, neural coding, and circuit dynamics.

If you like psychology

Model behaviour

Start with perception, attention, decision-making, reinforcement learning, drift diffusion models, and cognitive tasks.

If you like engineering / CS

Analyze brain data

Start with Python, signal processing, EEG/fMRI, machine learning, neural networks, and open datasets.

Best structured courses

Start with one structured course. Build notes, reproduce tutorials, and make one small project from it.

Best all-round start

Neuromatch Academy — Computational Neuroscience

Three-week immersive course, with hands-on tutorials and a global peer-learning community. Good if you want structure + coding notebooks + community.

  • Good for: beginners to intermediate learners
  • Needs: Python basics, math basics, some neuroscience exposure
  • Use it for: a guided entry into the field

Excellent for single-neuron models, network dynamics, decision models, and mathematical foundations. Includes online book, lectures, Python exercises, and teaching material.

Theoretical Neuroscience — Dayan & Abbott

A standard textbook for mathematical and computational modelling of neural systems. Use slowly, chapter-by-chapter, once you are comfortable with the basics.

  • Good for: graduate-level depth
  • Use it for: coding/decoding, networks, learning, decision models

What to learn before / alongside courses

Minimum starter stack: basic neuroscience + Python + Jupyter + linear algebra + probability/statistics + plotting + one project. You do not need to master all of calculus or machine learning before starting.
Neuroscience basics: neurons, synapses, action potentials, sensory systems, learning, memory.
Python: variables, loops, functions, arrays, notebooks, reading files, plotting.
Math: vectors, matrices, eigenvalues intuition, derivatives, simple differential equations.
Statistics: distributions, regression, correlation, hypothesis testing, confidence intervals.
Signal processing: filtering, Fourier transform, time-series, noise, sampling rate.
ML basics: train/test split, classification, regression, cross-validation, overfitting.

Project ideas that actually help

Use these for GitHub, SOPs, RA applications, internship emails, or interviews.

Beginner projects

  • Simulate an integrate-and-fire neuron and show firing rate vs input current.
  • Reproduce one Neuromatch tutorial and write a short explanation in your own words.
  • Analyze a simple behavioural dataset: RT, accuracy, regression, plots.
  • Use OpenNeuro sample data and make a basic brain-imaging visualization.

Intermediate projects

  • Build a drift diffusion model explanation with simulated reaction-time data.
  • Use MNE-Python to preprocess an EEG dataset and plot event-related potentials.
  • Use scikit-learn for basic fMRI/EEG decoding with cross-validation.
  • Train a simple neural network to model a perceptual decision task.

Advanced projects

  • Compare encoding vs decoding approaches on an open brain dataset.
  • Implement a reinforcement learning model for decision-making behaviour.
  • Fit multiple computational models and compare AIC/BIC or cross-validated performance.
  • Write a short report like a mini-paper: question, method, result, limitation.

What to put on GitHub

  • README: one-paragraph goal + figure preview.
  • Clean notebook with comments.
  • Data source and citation.
  • Final plots and a short interpretation.
  • One “what I learned” section.

A simple 12-week plan

1

Weeks 1–2: Foundations

Python basics, Jupyter, NumPy, plotting, basic neuron/synapse concepts. Goal: one notebook with clean plots.

2

Weeks 3–4: One structured course

Start Neuromatch or UW Coursera. Do not just watch — run notebooks and write your own notes.

3

Weeks 5–6: Modelling basics

Simulate integrate-and-fire neurons, tuning curves, simple neural networks or decision models.

4

Weeks 7–8: Open data

Pick OpenNeuro, DANDI or Allen data. Ask one question and make 2–3 publication-style plots.

5

Weeks 9–10: Mini-project

Write a small report: question, method, result, what it means, limitations. Push code to GitHub.

6

Weeks 11–12: Apply / email labs

Use the project as proof of seriousness. Email 5–10 labs with a tailored message.

Common mistakes to avoid

Only watching courses

Watching feels productive, but labs want evidence. Run code. Make plots. Write short reports.

Trying to learn everything

You do not need all of ML, all of neuroscience, and all of math. Choose one entry point and build depth slowly.

No visible proof

A small GitHub project or report can make your email much stronger than “I am interested in neuroscience.”
